## Les Étapes

# 1. Installation

## Dossier Frontend
```
npm install
```

## Dossier Backend
```
npm install

Dépendences
npm  async
npm  bcrypt
npm  body-parser
npm  express
npm  jsonwebtoken
npm  multer
npm  mysql2
npm  sequelize

```

## 2. Base de données
```
utiliser Xamp ou Mamp
sur le port 3306 pour mysql
npm run init dans le Backend
```

## 3. Démarrage
```
nodemon server
```

Le serveur est lancé sur le port 3001
Lancer le fichier index.html dans le navigateur
